Micro String Wound Filter

PRODUCT OVERVIEW

Mechmann's Micro String Wound Filter Cartridges are designed for superior performance in various industrial processes. By using a structured winding pattern, we achieve a gradient density that allows for deeper particle penetration and extended cartridge life.

MEDIA PP / Cotton / GF
MICRON 0.5 - 150 µm
LENGTHS 10" to 50"
✔ Gradient Density
✔ FDA Compliant
✔ Thermal Stability
✔ Low Pressure Drop
Core Material Polypropylene, SS304, SS316, Tinned Steel
Outer Diameter 60mm, 63mm, 115mm (Big Blue)
Micron Rating 0.5, 1, 5, 10, 25, 50, 75, 100, 150
Max Temp. PP: 80°C, Cotton: 120°C, GF: 400°C
Certifications ISO 9001:2008 / FDA Compliant
